# 6/25 Grand Canyon South Rimツアー

6/25(木)はGrand Canyon South Rimツアーに行く前提で固定します。前日6/24はTag Team、翌日6/26はフリー日なので、6/25は「観光に振り切る日」として扱い、夜のポーカー/ショーは基本なし。

## 結論

| 項目 | 内容 |
| --- | --- |
| 決定 | 6/25(木) Grand Canyon South Rimツアー |
| 1人予算目安 | `$150-$280` + チップ/軽食 |
| 拘束時間 | 約14-15時間 |
| 内容 | Grand Canyon National Park側の展望、複数ビューポイント、写真 |
| 夜の予定 | 原則なし。帰着後は軽い食事と休息 |
| 注意 | 朝が早く、帰りが遅い。翌6/26の昼は軽めにする |

## South Rimバス

| 項目 | 内容 |
| --- | --- |
| 目安費用 | `$120-$220` + チップ/軽食 |
| 所要時間 | 約14-15時間 |
| できること | Grand Canyon National Park側の展望、複数ビューポイント、写真 |
| 良い点 | 一般的に「国立公園のGrand Canyon」として期待する景色に近い |
| 注意 | 遠い。ほぼ移動日。翌日のフリー日は軽めに始める |

当日イメージ:

| 時間 | 内容 |
| --- | --- |
| 05:30-06:30 | ホテル集合/出発 |
| 06:30-12:30 | 移動 |
| 12:30-15:30 | South Rim観光 |
| 15:30-21:00 | ラスベガスへ戻る |
| 夜 | ほぼ寝るだけ |

向いている場合:

- 体験の質を優先し、長時間移動を許容できる。
- 「せっかくなら国立公園側を見たい」が強い。
- 6/25夜のポーカーやショーは完全に捨てられる。
