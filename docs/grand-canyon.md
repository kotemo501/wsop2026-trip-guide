# 6/25 Grand Canyonツアー案

6/25(木)をGrand Canyonツアーに使う場合の検討ページです。前日6/24はTag Team、翌日6/26は19:00フリーロールなので、Grand Canyonを入れるなら「6/25は観光に振り切る日」として扱います。

## 結論

| 案 | 1人予算目安 | 拘束時間 | 内容 | 判断 |
| --- | ---: | --- | --- | --- |
| A: West Rimヘリ | `$500-$750` | 約3.5-5時間 | Hoover Dam、Lake Mead、Grand Canyon Westを空から見る。プランにより谷底着陸 | 体験重視なら本命 |
| B: West Rimバス | `$130-$240` | 約11-12時間 | Eagle Point、Guano Point、Skywalk追加、写真 | 安く行くなら候補 |
| C: South Rimバス | `$150-$280` | 約14-15時間 | Grand Canyon National Park側の展望 | 景色重視だがかなり重い |
| D: 行かない | `$50-$250` | 半日-夜 | Sphere、ショー、AREA15、Fremontなどに回す | 疲労を避ける案 |

今の方針が「食事より体験にお金をかけたい」なら、Grand Canyonに行く場合はAのWest Rimヘリが一番合う。バスは安いが、6/25をほぼ全部使うので、6/25夜のポーカーやショーは基本なしで考える。

## まず決めること

| 決めること | 選択肢 | おすすめ |
| --- | --- | --- |
| Grand Canyonに行くか | 行く / 行かない | 行くなら6/25に固定 |
| 移動手段 | ヘリ / バス / レンタカー | ヘリ or バス。レンタカーは負担が大きい |
| 行き先 | West Rim / South Rim | 時間重視ならWest Rim、景色重視ならSouth Rim |
| 予算上限 | `$250` / `$600` / `$900+` | 体験重視なら`$600`前後 |
| 夜の予定 | なし / 軽い食事 / キャッシュ少し | 原則なし。ヘリなら夜に余力あり |

## 候補A: West Rimヘリ

| 項目 | 内容 |
| --- | --- |
| 目安費用 | `$439-$650`前後 + チップ/写真等 |
| 所要時間 | 約3.5-5時間 |
| できること | Hoover Dam、Lake Mead、Grand Canyon West上空、プランにより谷底着陸 |
| 良い点 | 時間効率が高い。体験として強い。夜に余力が残りやすい |
| 注意 | 高い。天候・体重制限・キャンセル条件を確認する |

当日イメージ:

| 時間 | 内容 |
| --- | --- |
| 午前 or 昼 | ホテル送迎/ターミナル集合 |
| 3.5-5時間 | ヘリ観光。プランにより着陸、軽食、シャンパン等 |
| 午後-夕方 | ホテル帰着 |
| 夜 | 軽い食事、Fremont、Sphere外観、短時間キャッシュなど |

向いている場合:

- 食事より体験にお金をかけたい。
- 6/25をGrand Canyonだけで潰したくない。
- 翌日のフリーロールに疲れを残しすぎたくない。

## 候補B: West Rimバス

| 項目 | 内容 |
| --- | --- |
| 目安費用 | `$105-$180` + チップ/軽食/オプション |
| 所要時間 | 約11-12時間 |
| できること | Hoover Dam周辺、Eagle Point、Guano Point、Skywalk追加、写真 |
| 良い点 | 安い。ホテル送迎付きが多い。West Rimはラスベガスから近い |
| 注意 | 朝早く、帰りも遅い。6/25夜の予定は入れにくい |

当日イメージ:

| 時間 | 内容 |
| --- | --- |
| 06:30-07:30 | ホテル集合/ピックアップ |
| 07:30-10:30 | 移動、Hoover Dam周辺の写真休憩 |
| 10:30-14:30 | West Rim観光。Eagle Point、Guano Point、昼食 |
| 14:30-18:30 | ラスベガスへ戻る |
| 夜 | 軽い食事、休息。キャッシュは打っても短時間 |

向いている場合:

- Grand Canyonには行きたいが、ポーカー予算を削りたくない。
- ヘリに`$500+`を出すほどではない。
- 長時間バスでも大丈夫。

## 候補C: South Rimバス

| 項目 | 内容 |
| --- | --- |
| 目安費用 | `$120-$220` + チップ/軽食 |
| 所要時間 | 約14-15時間 |
| できること | Grand Canyon National Park側の展望、複数ビューポイント、写真 |
| 良い点 | 一般的に「国立公園のGrand Canyon」として期待する景色に近い |
| 注意 | 遠い。ほぼ移動日。翌日のフリーロールに疲れが残る |

当日イメージ:

| 時間 | 内容 |
| --- | --- |
| 05:30-06:30 | ホテル集合/出発 |
| 06:30-12:30 | 移動 |
| 12:30-15:30 | South Rim観光 |
| 15:30-21:00 | ラスベガスへ戻る |
| 夜 | ほぼ寝るだけ |

向いている場合:

- 体験の質を優先し、長時間移動を許容できる。
- 「せっかくなら国立公園側を見たい」が強い。
- 6/25夜のポーカーやショーは完全に捨てられる。

## 予算の置き方

| 案 | ツアー | チップ/軽食/写真 | 合計目安 |
| --- | ---: | ---: | ---: |
| West Rimバス | `$105-$180` | `$25-$60` | `$130-$240` |
| South Rimバス | `$120-$220` | `$25-$60` | `$150-$280` |
| West Rimヘリ | `$439-$650` | `$40-$100` | `$500-$750` |
| West Rimヘリ全部盛り | `$700-$900+` | `$50-$120` | `$800-$1,050+` |
