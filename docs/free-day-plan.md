# 6/26フリー日

Grand Canyon翌日の6/26(金)は調整日。昼は回復、買い物、近場散歩を軸にし、体力が残っていれば19:00から参加受付のフリートーナメントを候補にする。

## 朝の判断

| 朝の状態 | 今日の動き |
| --- | --- |
| 眠い、足が重い、前日帰着が遅い | 回復日。水、軽食、洗濯、荷物整理 |
| 普通に動ける | 買い物を1か所だけ入れる |
| かなり元気 | Bellagio/Caesars/Venetian周辺を短く散歩。夜にフリートーナメント受付へ行くか判断 |

迷ったら回復日。6/27はSphereと空港移動があるので、6/26夜に疲れを残さない。

## 1日の流れ

| 時間 | 動き |
| --- | --- |
| 午前 | 起きる時間を固定しない。水、軽食、洗濯、荷物整理 |
| 昼 | Horseshoe周辺で買い物、水・軽食補充 |
| 夕方 | 余裕があれば近場を短く散歩 |
| 19:00 | フリートーナメント参加受付。体力がある場合だけ候補 |
| 夜 | 参加しない/終了後は、6/27の荷物、Sphereチケット、空港集合導線を確認 |

## やらないこと

- 遠出。
- 夜まで長引く予定。
- Grand Canyon疲れが残っているのにフリートーナメントへ行くこと。
- 最終日の荷造りを後回しにすること。
- 体力が落ちている状態で予定を増やすこと。

## 朝のチェック

- Grand Canyon疲れは残っていないか。
- 現金とカードは分けて持てているか。
- 6/27のSphereチケット、荷物預け、空港集合の確認は済んでいるか。
- 買い物をするなら帰りの荷物重量に余裕があるか。
- 19:00受付に行くなら、パスポート、スマホ、カード、水、上着を持ったか。
