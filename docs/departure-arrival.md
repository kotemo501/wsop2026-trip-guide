# 出発・到着導線

成田出発からHorseshoe到着、ウェルカムパーティーまでの初日メモ。チャーター便の正確な集合時刻、空港カウンター、現地送迎はツアー最終案内を優先する。

## 目的

- 出発当日に、パスポート、荷物、現金、スマホで詰まらない。
- 機内で必要なものを預け荷物に入れない。
- ラスベガス到着後は、ホテル導線、水、軽食、WSOP会場位置だけ確認して無理しない。

## 出発前夜

| 確認 | 完了条件 |
| --- | --- |
| パスポート | 原本を機内持込へ |
| ESTA | 公式画面または控えをGitLab外で保存 |
| ツアー最終案内 | 空港集合、カウンター、荷物条件、現地送迎を確認 |
| 機内持込 | 2個合計7kg目安。超過しそうなら追加枠確認 |
| 預け荷物 | 30kg以下、3辺203cm以下目安。貴重品を入れない |
| モバイルバッテリー | 預けず機内持込 |
| 現金/カード | ポーカー用、生活用、Emergencyを分ける |
| 非公開控え | Grand Canyon、Sphere、保険、カード停止をGitLab外で開ける |

## 成田空港

| 順 | やること |
| --- | --- |
| 1 | ツアー最終案内の集合場所/カウンターへ行く |
| 2 | パスポート、必要書類、預け荷物を出せる状態にする |
| 3 | 機内持込の重量、液体、モバイルバッテリーを確認 |
| 4 | 現金とカードを預け荷物に入れていないか確認 |
| 5 | 搭乗前にスマホ、Wallet、非公開控え、オフライン資料を確認 |

空港で焦らないため、外貨両替や買い足しは時間に余裕がある時だけにする。

## 機内手元品

| 必須 | 理由 |
| --- | --- |
| パスポート | 入国、本人確認 |
| 現金、クレカ、予備カード | 預け荷物に入れない |
| スマホ、充電器、モバイルバッテリー | 到着後の連絡、地図、予約確認 |
| 薬、コンタクト、メガネ | ロストバゲージ対策 |
| 1日分の着替え | 預け荷物遅延対策 |
| 羽織りもの | 機内とカジノの冷房対策 |
| 予約/保険の非公開控え | 通信不良時の保険 |

## ラスベガス到着後

| 順 | やること |
| --- | --- |
| 1 | 入国、荷物受け取り、税関を落ち着いて通過 |
| 2 | ツアー側の送迎/集合導線に従う |
| 3 | Horseshoe Las Vegasへ移動 |
| 4 | チェックイン可否、荷物預け、Bell Desk位置を確認 |
| 5 | 水、軽食、カフェイン、日用品を調達 |
| 6 | WSOP会場、Rideshare Lot、Bell Deskの位置を軽く確認 |
| 7 | 18:00-21:00 Ellis Island Casinoウェルカムパーティーへ |

到着日は長時間キャッシュや遠い観光を入れない。翌日6/23のDailyと6/24 Tag Teamに響かない動きを優先する。

## 初日に確認する場所

Horseshoe内外の移動優先順位は [onsite-navigation-card.md](onsite-navigation-card.md) にまとめる。

| 場所 | 理由 |
| --- | --- |
| Horseshoe Bell Desk | 6/27荷物預け候補 |
| WSOP会場導線 | 6/23 Daily、6/24 Tag Teamで迷わない |
| Rideshare Lot / North Entrance | 6/25 Grand Canyon pickupの下見 |
| CVS/Walgreensなど | 水、軽食、日用品の調達 |
| Ellis Island Casino | ウェルカムパーティー会場 |

## やらないこと

- 到着日から長時間キャッシュ。
- 深夜までの遠い観光。
- 眠い状態での大きい買い物。
- パスポート、現金、カードを同じバッグに集約すること。
