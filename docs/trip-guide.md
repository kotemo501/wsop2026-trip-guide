# WSOP 2026 旅のしおり原稿

このファイルをWebページ化の元原稿にします。細かい調査メモは各詳細ファイルに残し、このページは友人と見ながら意思決定できる粒度に絞ります。

## 使い方

- 出発前: 「未決事項」と「出発前ToDo」を上から潰す。
- 相談時: 「日別プラン」と「相談すること」を見ながら、6/26や予算上限を決める。
- 現地: 「日別プラン」「お金」「荷物FAQ」だけ見れば動ける状態にする。
- Webページ: この原稿をもとに、情報をタブやカードに分けて表示する。

## 日別プラン

### 6/22(月) 到着日

メイン:

- 成田発、ラスベガス着。
- ホテルチェックイン。
- 18:00-21:00 ウェルカムパーティー参加。

この日にやること:

- 水、軽食、カップ麺などを買う。
- WSOP会場やホテル周辺の導線を軽く確認する。
- 翌日のDailyに備えて無理しない。

避けること:

- 到着日から長時間キャッシュ。
- 遠い観光。

### 6/23(火) Daily Deepstack参加日

メイン:

- 13:00 `$250 Daily Deepstack NLH`に参加。

飛んだ後の候補:

| 候補 | 内容 | 先に決めること |
| --- | --- | --- |
| キャッシュ | 1-2セッションだけ | 損失上限、終了時刻、追加Buy-in可否 |
| 休息 | 軽食、散歩、早寝 | 翌日のTag Team優先 |

おすすめ:

- キャッシュ上限はまず`$300`か`$500`で相談。
- 翌日Tag Teamがあるので、深夜まで打たない。

### 6/24(水) Tag Team参加日

メイン:

- 12:00 WSOP Event #66 `$1,000 NLH Tag Team`。
- 2人で1チーム。支払いは1人`$500`想定。

確認すること:

- 登録方法。
- 集合時間。
- 交代ルール。
- 支払い方法。
- パスポートやIDの扱い。

夜:

- 勝ち残りや疲労次第。
- 敗退後にもう1本入れるより、体力を優先する。

### 6/25(木) Grand Canyon South Rim

メイン:

- South Rimツアーで固定。
- 約14-15時間の終日観光。

当日のイメージ:

| 時間帯 | 内容 |
| --- | --- |
| 早朝 | 集合、出発 |
| 午前 | 移動 |
| 昼-午後 | South Rim観光、写真 |
| 夜 | ラスベガス帰着、軽食、休息 |

持ち物:

- 水、軽食。
- 薄手の上着。
- 歩きやすい靴。
- サングラス、帽子、日焼け止め。
- モバイルバッテリー。
- チップ用現金。

### 6/26(金) フリー日

目的:

- Grand Canyon翌日の回復日。
- お土産、散歩、キャッシュ、トーナメントの調整日。

候補:

| 候補 | 予算 | 向いている場合 |
| --- | ---: | --- |
| お土産・近場散歩 | `$0-$250` | 疲れている、写真や買い物を回収したい |
| 13:00 Daily | `$250` | もう1本トーナメントを打ちたい |
| 19:00 フリーロール | `$0` | 無料イベントを楽しみたい |
| キャッシュ | `$300-$500`上限 | 時間を自分たちで決めたい |
| 他カジノ見学 | `$0-$80` | Bellagio / Aria / Wynnなどを見たい |

先に決めること:

- フリー日の優先順位を「休息」「観光」「ポーカー」のどれに寄せるか。
- キャッシュを打つ場合の損失上限。

### 6/27(土) 最終日

メイン:

- チェックアウト。
- 14:00 Sphere公演。
- 他カジノ見学。
- 夜に空港へ。

この日にポーカーを入れない理由:

- 帰国便固定。
- 荷物管理と空港移動がある。
- トーナメントは時間が読みにくい。

確認すること:

- チェックアウト後に荷物をどこに預けるか。
- Sphereの公演名、座席、入場時刻。
- Sphereからホテル/集合場所/空港への移動時間。
- 空港集合時刻。
