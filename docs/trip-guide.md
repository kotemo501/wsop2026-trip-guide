## 日別プラン

### 6/22(月) 到着日

- 成田発、ラスベガス着。
- ホースシュー・ラスベガスにチェックイン。
- 18:00-21:00 Ellis Island Casinoウェルカムパーティー参加。
- 水、軽食、カップ麺などを買う。
- WSOP会場やホテル周辺の導線を軽く確認する。
- 翌日のDailyに備えて無理しない。

### 6/23(火) Daily Deepstack

- 13:00 `$250 Daily Deepstack NLH`に参加。
- 朝にWSOP LIVE、Caesars Rewards、会場導線、支払い手段を確認。
- 終了後は翌日のTag Teamを優先し、登録場所、支払い、集合時刻を確認する。
- 深夜まで粘らない。

### 6/24(水) Tag Team

- 10:00目安でVersailles Ballroom方面へ行く。
- WSOP LIVE、Caesars Rewards、本人確認、支払いを確認。
- 12:00 WSOP Event #66 `$1,000 NLH Tag Team`に参加。
- 夜は6/25 Grand Canyonの集合場所、持ち物、起床時刻を確認する。

### 6/25(木) Grand Canyon South Rim

| 時間帯 | 内容 |
| --- | --- |
| 04:15 | 起床、身支度、カード/ID/水/上着/予約票確認 |
| 04:50 | 部屋を出る |
| 05:00 | Horseshoe Las Vegas Rideshare Lot到着 |
| 05:05 | pickup予定 |
| 05:15 | National Park Express出発 |
| 昼-午後 | South Rim |
| 夜 | ラスベガス帰着、軽食、休息 |

集合場所:

- Horseshoe Las Vegas Rideshare Lot。
- Flamingo Road側のNorth Entrance外。
- Real Bodies Exhibit近く。

支払い:

- ツアー代は予約済み。
- Grand Canyon National Parkの非米国居住者feeは`$100/人`を入場時にnon-cashで別途支払い。2人で`$200`。
- Guide/Driverへのチップは別。`$10-$20/人`目安。

### 6/26(金) フリー日

- Grand Canyon翌日の回復日。
- 午前は起床時刻を固定しない。
- 動く場合は買い物か近場散歩までに絞る。
- 夜は6/27の荷物、Sphereチケット、空港集合導線を確認する。

### 6/27(土) 最終日

- 昼前までにチェックアウトと荷物預け。
- Sphereへ大きいバッグを持って行かない。
- 13:15入場目安、14:00 The Wizard of Oz at Sphere。
- 公演後はホテルへ戻って荷物回収。
- 夜はツアー最終案内の空港集合場所と時刻を優先する。
