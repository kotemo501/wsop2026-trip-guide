## 予約方法メモ

### Grand Canyon South Rim

予約の進め方:

1. 6/25(木)に催行があるSouth Rimバスツアーを探す。
2. Horseshoe Las Vegas送迎、または徒歩圏の集合場所があるか確認する。
3. 集合時刻、帰着時刻、キャンセル条件、昼食、水、チップ、荷物制限を確認する。
4. 予約完了後、予約番号と集合場所をこのしおりに反映する。

調査時点の候補:

| 候補 | 見るポイント |
| --- | --- |
| GrandCanyon.com South Rim Bus Tour | 14-15時間、ホテル送迎、昼食込み、72時間キャンセル条件の記載あり |
| Gray Line系 South Rim Bus Tour | Strip/Downtown主要ホテル送迎、予約確認、キャンセル条件、荷物制限を確認 |
| Grand Canyon Tour Company / Scenic等 | 価格、集合場所、帰着時刻、バス設備で比較 |

予約前チェック:

- South Rimであること。West Rimではないこと。
- 集合場所がHorseshoeか徒歩圏か。
- 早朝集合に間に合うか。
- 大きな荷物やスーツケースを持ち込めない場合があるか。
- キャンセル期限。

### Sphere

予約の進め方:

1. Sphere公式サイトで6/27(土) 14:00の公演を確認する。
2. 公式サイトのBuy Tickets導線からTicketmaster等の販売ページへ進む。
3. 公演名、座席、総額、入場時刻、モバイルチケット、バッグポリシーを確認する。
4. 予約後、チケット表示方法と入場時刻をこのしおりに反映する。

確認ポイント:

- 公演名。
- 14:00開演で間違いないか。
- チケット総額。
- 座席位置。
- モバイルチケットの表示方法。
- 荷物制限。
- Sphereからホテル/空港集合場所までの移動時間。

## 飛行機・荷物FAQ

詳細は[flight-baggage.md](flight-baggage.md)。

ZIPAIRの一般ルールをベースにしたメモ。今回はチャーター便なので、最終案内で条件が変わる可能性がある。

| 項目 | 目安 |
| --- | --- |
| 機内持込 | 2個まで。合計7kg無料 |
| サイズ目安 | 40 x 25 x 55cm、35 x 25 x 45cm |
| 重量追加 | 合計15kgまでの追加枠あり。事前確認 |
| 受託手荷物 | 1個30kg超、3辺203cm超は不可 |
| 機内に入れる | パスポート、現金、カード、薬、充電器、モバイルバッテリー、1日分の着替え |
| 預ける | 衣類、洗面具、刃物類、液体類の予備 |

モバイルバッテリー/液体類:

- モバイルバッテリーや予備リチウム電池は預けず、機内持込にする。
- 液体類は国際線の一般ルールとして、機内持込は100ml以下の容器を透明袋に入れる前提で準備する。
- 刃物、工具、スプレー、ガス缶、火薬類は機内持込にしない。迷うものは預け荷物か、持って行かない。

## 海外旅行保険

アメリカは医療費が高いため、クレカ付帯だけで足りるかを確認し、不安なら任意加入を検討する。

候補:

| 候補 | 見るポイント |
| --- | --- |
| AIG損保 海外旅行保険 | 治療・救援費用の無制限プラン、キャッシュレス医療、歯科/既往症急変の扱い |
| t@biho たびほ | Web申込、補償額、航空機遅延、携行品、保険料 |
| クレカ付帯保険 | 利用付帯/自動付帯、傷害/疾病治療費用、救援費用、家族特約 |

確認すること:

- 傷害治療費用、疾病治療費用、救援者費用。
- キャッシュレス診療の有無。
- 航空機遅延、手荷物遅延、携行品損害。
- クレカ付帯の場合、旅行代金決済が必要か。
- 既往症や歯科の扱い。
