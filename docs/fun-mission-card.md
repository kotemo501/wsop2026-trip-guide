# 旅を楽しくするミッションカード

予定を増やすためではなく、旅の満足度を上げるための小さなミッション集。疲れている日は1つだけでよい。予約番号、個人情報、チケットQR、支払い情報は書かない。

## 使い方

1. 朝に今日の固定予定を見る。
2. 体力を [condition-care-card.md](condition-care-card.md) でGreen/Yellow/Redに分ける。
3. Greenなら2-3個、Yellowなら1個、Redなら休息ミッションだけ選ぶ。
4. 6/25 Grand Canyon、6/27空港導線に響く予定は増やさない。

## 共通ミッション

| 種類 | ミッション | 目安 |
| --- | --- | --- |
| 写真 | 今日のベスト1枚を2人で選ぶ | 1日1回 |
| 食事 | 「節約食でも旅っぽい」軽食を1つ見つける | `$5-$20` |
| ポーカー | 今日の良かった1ハンド/1判断をメモする | 3分 |
| 体力 | 水を買う、座る、上着を持つ | 毎日 |
| 記念 | カジノチップ、レシート、ショップ袋など公開してよい思い出を1つ残す | 任意 |

## 6/22(月) 到着日

目的: ラスベガスに来た感を作る。夜更かしより導線確認を優先。

- HorseshoeからWSOP会場までの行き方を一度歩く。
- Bellagio方面の夜景を短く見る。
- 水、軽食、翌朝のカフェインを買う。
- ウェルカムパーティーで「今回の旅で絶対やりたいこと」を1つずつ言う。

やめること:

- 長時間キャッシュ。
- 遠い観光。
- 翌日のDailyに響く夜更かし。

## 6/23(火) Daily Deepstack

目的: 初WSOPの雰囲気をしっかり味わう。

- WSOP会場の写真を撮る。
- 13:00 Daily前に、会場導線と登録/支払い場所を確認する。
- Daily後に、良かった判断を1つだけメモする。
- 敗退後に打つなら、終了時刻と損失上限を先に決める。

楽しい寄り道:

- Horseshoe/Paris周辺で短い写真散歩。
- 軽食を買って翌日のTag Team準備。

## 6/24(水) Tag Team

目的: 2人旅らしい本命イベントを楽しむ。

- 開始前に2人でチーム写真を撮る。
- 交代ルールを同じ理解にする。
- 「今日のチーム方針」を一言で決める。
- 終了後は結果より、翌朝04:15起床の準備を優先する。

やめること:

- 登録をぎりぎりにする。
- 敗退後に深夜まで予定を入れる。

## 6/25(木) Grand Canyon South Rim

目的: ポーカーから切り替えて、観光に振り切る。

- 05:00に集合場所で1枚写真を撮る。ただし車両/他人/予約票は写さない。
- South Rimで「ベスト景色」を1つ決める。
- 水と上着をケチらない。
- 帰着後は軽食と休息をミッションにする。

やめること:

- 予約票や予約番号が写る写真を公開場所へ置く。
- 夜にポーカーやショーを入れる。

## 6/26(金) フリー日

目的: 回復しながら、ベガスらしい思い出を回収する。

朝に選ぶ:

| 体力 | ミッション |
| --- | --- |
| Red | 起きる時間を固定しない。水、軽食、荷物整理 |
| Yellow | Bellagio/Caesars/Venetianのどれか1つだけ回る |
| Green | お土産、短時間キャッシュ、Daily/フリーロール候補から1つ選ぶ |

おすすめ:

- お土産候補を写真で記録して、買うかは夜に決める。
- 近場のカジノを1つだけ見学する。
- キャッシュを打つなら、開始前に終了時刻を決める。

## 6/27(土) Sphere + 空港

目的: 最終日の満足度を上げつつ、帰国導線を崩さない。

- 荷物を預けたあと、身軽な状態で写真を撮る。
- Sphere前にスマホチケット、カード、パスポート、充電を確認する。
- 公演後に「この旅のベスト3」を2人で言う。
- 荷物回収と空港集合を最優先にする。

やめること:

- Sphereに大きいバッグを持って行く。
- 公演後に遠出する。
- 空港集合時刻ぎりぎりまで写真や買い物を引っ張る。

## 帰国後

- 写真を日別に分ける。
- 使った金額をざっくり振り返る。
- よかった店、よかったプレイ、次回改善を3つだけ残す。
- 公開できない写真や予約情報はGitLabに置かない。
