## おすすめの組み合わせ

### バランス型

- 6/24(水): Tag Team
- 6/26(金): フリーロール
- Daily Deepstack: `$250`か`$400`を1-2本
- キャッシュゲーム: 1-2回
- 観光: 半日から1日確保

一番おすすめ。2人旅らしさとWSOP体験を両立しやすい。

### トーナメント多め

- 6/23(火): `$1,500` Freezeout
- 6/24(水): Tag Team
- 6/26(金): フリーロール
- 6/27(土): 原則プレイしない

結果次第で予定が崩れやすい。勝ち進んだ場合の観光キャンセルを許容する人向け。

### Mystery Millionsをどう扱うか

- Day 1で敗退する前提なら「WSOPの大規模イベントを体験する」候補にはなる。
- Day 2に進むと2026-06-28(日) 11:00再開、Day 3は2026-06-29(月)なので、今回の帰国便とは両立しない。
- 旅程を守るなら、Mystery MillionsよりTag Team、Freezeout、Daily Deepstack、キャッシュを優先する。

### 低額トーナメントをどう入れるか

- まず候補にするのは13:00の`$250 Daily Deepstack`と16:00の`$400 Daily Deepstack`。
- 20:00の`$200 Daily Deepstack`は夜に軽く打ちたい時の候補。ただし翌日の予定に響きやすい。
- 6/26(金)は19:00フリーロール参加なので、16:00以降のDailyは原則避ける。
- 6/27(土)は空港移動があるので、Dailyでも見送り寄り。

### 観光多め

- 6/24(水): Tag Team
- 6/26(金): フリーロール
- キャッシュゲーム: 夜に短時間
- 6/25(木): 観光メイン

初ラスベガスならこちらも現実的。
