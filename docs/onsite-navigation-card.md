# 現地移動カード

Horseshoeを拠点に、WSOP、Grand Canyon集合、Sphere、空港集合で迷わないための移動カード。予約番号、参加者名、確認メール本文、個人の連絡先は書かない。ツアー最終案内や予約票と違う場合は、そちらを優先する。

## 拠点

| 場所 | 役割 |
| --- | --- |
| Horseshoe Las Vegas | ホテル拠点、WSOP、Grand Canyon pickup、荷物預け候補 |
| Bell Desk | 6/27荷物預け候補。最終案内または現地で確認 |
| Rideshare Lot / North Entrance | 6/25 Grand Canyon pickup下見 |
| WSOP会場導線 | 6/23 Daily、6/24 Tag Teamで使う |

## 初日に確認する4か所

1. 部屋からBell Desk。
2. 部屋からWSOP会場。
3. 部屋からRideshare Lot / North Entrance。
4. Horseshoe周辺の水・軽食調達場所。

## 6/25 Grand Canyon集合

| 項目 | 内容 |
| --- | --- |
| 集合時刻 | 05:05 |
| ツアー開始 | 05:15 |
| 集合 | Horseshoe Las Vegas Rideshare Lot |
| 場所メモ | Flamingo Road側North Entrance外、Real Bodies Exhibit近く |
| 部屋を出る目安 | 04:50 |
| 到着目安 | 05:00 |
| 前夜 | 実際に歩いて場所を確認する |

歩き方:

1. Horseshoe内で`Rideshare Lot`の標識を探す。
2. North Entranceから外へ出る。
3. Flamingo Road側、High Roller Ferris Wheel方向、The Cromwell/Flamingo方面を確認する。
4. Real Bodies Exhibit近くの入口を目印にする。
5. 前夜に写真を撮るなら、個人情報や予約票が写らないようにする。

遅れそうなら、まず相方へ連絡し、非公開の予約確認メールから会社へ連絡する。

## 6/27 Sphere + 空港

詳細は [final-day-luggage-airport-card.md](final-day-luggage-airport-card.md) も見る。

| 時刻目安 | 移動 |
| --- | --- |
| 昼前 | チェックアウト。荷物はHorseshoe Bell Deskまたはツアー指定場所へ |
| 12:45 | Sphereへ移動開始 |
| 13:15 | Sphere入場列へ |
| 14:00 | The Wizard of Oz at Sphere |
| 15:30以降 | Horseshoeへ戻り荷物回収 |
| 夜 | ツアー最終案内の空港集合場所・時刻を優先 |

判断:

- Sphereへ大きいバッグを持って行かない。
- 公演後は写真や買い物より荷物回収を優先する。
- 空港集合時刻が近いなら、観光を切って移動する。
- 徒歩かUber/Lyftかは、暑さ、疲労、荷物、集合時刻で決める。

## 迷った時の優先順位

1. ツアー最終案内。
2. 予約票/確認メールの非公開控え。
3. 公開しおりの集合時刻と場所。
4. 地図アプリ。
5. ホテル/施設スタッフへの確認。

## やらないこと

- 初見の集合場所へ当日ぎりぎりに向かう。
- Sphereへスーツケースや大きいバッグを持って行く。
- 最終日に遠い買い物を増やす。
- 空港集合時刻が近いのに写真や土産を優先する。
