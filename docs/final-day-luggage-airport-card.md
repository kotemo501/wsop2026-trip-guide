## 預け荷物に入れるもの

- スーツケース。
- 大きいバックパック。
- 液体類、刃物類、工具類。
- 買い物袋やお土産。
- 使わない衣類。

預けないもの:

- パスポート。
- スマホ、モバイルバッテリー。
- 現金全額。
- メインカード/予備カード全部。
- Sphereチケット表示に必要な端末。

## 判断ルール

| 状況 | 判断 |
| --- | --- |
| 荷物預け先がBell Deskとツアー指定で分かれる | ツアー最終案内を優先。迷うならBell Deskでなくツアー側へ確認 |
| Sphere後に時間が余る | 遠出しない。荷物回収、軽食、トイレ、充電 |
| 空港集合まで90分未満 | 観光を止めて荷物回収と移動へ |
| 徒歩で戻るか迷う | 暑さ、疲労、荷物、集合時刻でUber/Lyftを使う |
| チケット/Walletが開かない | Sphereへ行く前に解決。入場列で初確認しない |
