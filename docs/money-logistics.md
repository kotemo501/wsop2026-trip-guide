# お金の持って行き方

現地予算は[現地予算](budget.md)で、標準`$2,850`前後を想定します。ただし全部を現金で持つ必要はない。現金、クレカ、デビット/海外ATM、ポーカー用資金を分ける。

現地で朝に見る財布配分だけなら [daily-money-card.md](daily-money-card.md) を使う。

## 結論

| 用途 | おすすめ |
| --- | --- |
| 現金 | 1人`$2,000`前後を基本線。厚めでも`$2,500`程度まで |
| クレカ | Visa/Master系を2枚。Grand Canyon、Sphere、ホテル周辺支払い用 |
| デビット/海外ATM | 予備。ATM手数料が高いので常用しない |
| ポーカー用 | Daily/Tag Team/キャッシュゲーム分を別封筒 or 別財布で管理 |
| 日常財布 | 1日`$100-$200`だけ持ち歩く |

出発前の推奨結論:

- USD現金は標準`$2,000`前後を第一候補にする。
- 内訳は、トーナメント確定`$750`、キャッシュゲーム用`$900-$1,200`、チップ/小口/生活予備`$300-$400`を目安にする。
- クレカはVisa/Master系を中心に2枚以上。1枚は日常財布、1枚は予備として分ける。
- Grand Canyon非米国居住者feeはnon-cash対応、Sphereとホテル周辺支払いはカードを使える状態にする。
- WSOP/ポーカー資金は生活費と混ぜない。
- Emergency現金と予備カードは、持ち歩き財布と同じ場所に入れない。

日本出国時は、現金・小切手等の合計が100万円相当を超えると税関申告が必要。`$1 = 160円`なら、100万円は約`$6,250`。米国入国/出国は`$10,000`超でCBP/FinCEN申告が必要。申告すれば違法ではないが、今回の旅では申告ラインを超える現金を持つ必要はない。

## 持って行く金額案

| 案 | USD現金 | カード/ATM枠 | 向いている人 |
| --- | ---: | ---: | --- |
| 軽め | `$1,500-$1,700` | `$1,500+` | キャッシュゲームをかなり抑える |
| 標準 | `$1,900-$2,100` | `$2,000+` | Tag Team、Daily、`$1/$3`を300-400BB見る |
| ポーカー厚め | `$2,300-$2,500` | `$2,000+` | `$1/$3`を400BB超で見る |

おすすめは標準案。現金`$2,000`前後なら、トーナメント確定`$750`、`$1/$3`キャッシュゲーム300-400BB、チップ用現金を確保しつつ、日本の100万円申告ラインからもかなり余裕がある。

## 現金の分け方

| 封筒/財布 | 金額目安 | 用途 |
| --- | ---: | --- |
| Poker固定 | `$750` | 6/23 Daily `$250` + Tag Team割り勘 `$500` |
| Cash Game | `$900-$1,200` | `$1/$3`想定。300-400BB |
| Tips/交通/軽食 | `$200-$300` | ツアーチップ、Uber、軽食、買い出し |
| Emergency | `$300-$500` | 使わない予備 |
| Daily wallet | `$100-$200` | その日持ち歩く分 |

ホテルのセーフティボックスが使えるなら、持ち歩く現金は少なくする。カジノやイベント会場では財布を分け、ポーカー用資金と日常支払いを混ぜない。

## カード構成

| カード | 役割 | 持ち方 |
| --- | --- | --- |
| メインカード | Grand Canyon fee、Sphere、買い物、ホテル周辺支払い | 日常財布。利用通知ON、暗証番号確認 |
| 予備カード | メイン停止時の食事、移動、空港対応 | 別ポーチまたはホテル保管。日常財布と同居させない |
| ATM/デビット | 現金不足時の最後の予備 | 手数料、限度額、暗証番号、海外利用設定を事前確認 |

カード番号、停止窓口、保管場所の詳細はGitLabに書かない。[safety-wallet.md](safety-wallet.md) の非公開控えに入れる。

## 支払い手段ごとの使い分け

| 手段 | 使う場面 | 注意 |
| --- | --- | --- |
| USD現金 | ポーカー、チップ、軽食、緊急時 | 多額を持ち歩かない |
| クレカ | Sphere、Grand Canyon、買い物、ホテル周辺 | 海外利用停止・限度額・暗証番号を確認 |
| デビット/プリペイド | ATM予備、カード分散 | 為替手数料・ATM手数料を確認 |
| 海外ATM | 現金不足時 | カジノATMは手数料が高い可能性 |

## WSOP/ポーカー支払い

| 項目 | 方針 |
| --- | --- |
| 6/23 Daily | 現金`$250`をすぐ出せるようにする |
| 6/24 Tag Team | 2人で支払い分担を事前に決める。1人`$500`想定 |
| トーナメント確定分 | 合計`$750`。カード/WSOP LIVE支払いもあり得るが、現金でも払える状態にする |
| キャッシュゲーム | `$1/$3`想定。300-400BBなら`$900-$1,200` |

2026 WSOP公式登録ページでは、現地buy-inはcashを受け付ける。オンラインのcredit/debit card登録は初回の現地verificationが必要で、Visa/Mastercard/Discover/Amex利用時は追加3% feeの記載がある。したがって、トーナメントはカード決済だけに寄せず、現金でも払える状態にしておく。

## 申告ライン

| 場面 | 申告ライン | 今回の方針 |
| --- | ---: | --- |
| 日本出国/入国 | 100万円相当超 | USD現金は`$6,000`未満に抑える |
| 米国入国/出国 | `$10,000`超 | 持ち込まない |

米国の`$10,000`は、現金だけでなくトラベラーズチェック、マネーオーダー、一定の小切手等も含む。家族などグループで旅行している場合、合計額で判定されることがある。日本側も現金・小切手・有価証券等の合計で100万円相当超なら申告対象。

## 両替・外貨宅配

| 方法 | 向いている場合 | 注意 |
| --- | --- | --- |
| 外貨宅配 | 出発前に余裕を持って準備したい、レート比較したい | 配送日数、送料、最低金額、本人確認、受取方法を確認 |
| 空港両替 | 当日受け取りで安心したい | レートは事前比較。混雑時は時間に余裕を見る |
| 銀行/街中両替 | 近くに店舗がある | 営業時間、米ドル在庫、手数料を確認 |
| 現地ATM | 不足時の予備 | カジノATMは手数料が高い可能性。常用しない |

おすすめは、出発1-2週間前に外貨宅配または国内店舗で`$2,000`前後を用意し、不足時だけカード/ATMで補う形。外貨宅配はレートと手数料を比較しやすいが、郵送の受取日数と本人確認があるため直前には向かない。

コスパ重視ならネット外貨宅配を第一候補にする。GPA Onlineは安心感、インターバンク/外貨両替ドルユーロ/SBI FXトレード系はレート面で比較対象。送料込みの実質レートで見る。

## 現地でのルール

- ポーカー資金と生活費を混ぜない。
- キャッシュの損失上限は各自で決める。
- 酒を飲んだらキャッシュの追加Buy-inをしない。
- ATMで下ろす前に、まず残りの現金と翌日の予定を確認する。
- 盗難/紛失時に備えて、カード会社の停止方法をメモしておく。

## 帰国日前のルール

- 6/27は買い物より荷物回収と空港集合を優先する。
- 予備カードとEmergency現金は最後まで残す。
- Sphere後に財布を広げて現金整理しない。ホテル戻り後か空港到着後にする。
- 使い切るためのキャッシュゲームはしない。
