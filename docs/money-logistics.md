# お金の持って行き方

現地予算は[現地予算](budget.md)で、標準`$2,700`前後を想定します。ただし全部を現金で持つ必要はない。現金、クレカ、デビット/海外ATM、ポーカー用資金を分ける。

## 結論

| 用途 | おすすめ |
| --- | --- |
| 現金 | 1人`$1,500-$2,500`を目安。`$6,000+`は避ける |
| クレカ | Visa/Master系を2枚。Grand Canyon、Sphere、ホテル周辺支払い用 |
| デビット/海外ATM | 予備。ATM手数料が高いので常用しない |
| ポーカー用 | Daily/Tag Team/キャッシュ候補分を別封筒 or 別財布で管理 |
| 日常財布 | 1日`$100-$200`だけ持ち歩く |

日本出国時は、現金・小切手等の合計が100万円相当を超えると税関申告が必要。`$1 = 160円`なら、100万円は約`$6,250`。米国入国/出国は`$10,000`超でCBP/FinCEN申告が必要。申告すれば違法ではないが、今回の旅では申告ラインを超える現金を持つ必要はない。

## 持って行く金額案

| 案 | USD現金 | カード/ATM枠 | 向いている人 |
| --- | ---: | ---: | --- |
| 軽め | `$1,200-$1,500` | `$1,500+` | キャッシュ少なめ、カード中心 |
| 標準 | `$1,800-$2,200` | `$2,000+` | Tag Team、Daily、キャッシュ少し |
| ポーカー厚め | `$2,500-$3,000` | `$2,000+` | `$1/$2`キャッシュを複数回打つ可能性を見る |

おすすめは標準案。現金`$2,000`前後なら、ポーカーの初動資金とチップ用現金を確保しつつ、日本の100万円申告ラインからもかなり余裕がある。

## 現金の分け方

| 封筒/財布 | 金額目安 | 用途 |
| --- | ---: | --- |
| Poker固定 | `$750` | 6/23 Daily `$250` + Tag Team割り勘 `$500` |
| Cash Game候補 | `$300-$500` | 6/23敗退後や6/26の`$1/$2`キャッシュ |
| Tips/交通/軽食 | `$200-$300` | チップ、Uber、軽食、買い出し |
| Emergency | `$300-$500` | 使わない予備 |
| Daily wallet | `$100-$200` | その日持ち歩く分 |

ホテルのセーフティボックスが使えるなら、持ち歩く現金は少なくする。カジノやイベント会場では財布を分け、ポーカー用資金と日常支払いを混ぜない。

## 支払い手段ごとの使い分け

| 手段 | 使う場面 | 注意 |
| --- | --- | --- |
| USD現金 | ポーカー、チップ、軽食、緊急時 | 多額を持ち歩かない |
| クレカ | Sphere、Grand Canyon、買い物、ホテル周辺 | 海外利用停止・限度額・暗証番号を確認 |
| デビット/プリペイド | ATM予備、カード分散 | 為替手数料・ATM手数料を確認 |
| 海外ATM | 現金不足時 | カジノATMは手数料が高い可能性 |

## WSOP/ポーカー支払い

| 項目 | 方針 |
| --- | --- |
| 6/23 Daily | 現金`$250`をすぐ出せるようにする |
| 6/24 Tag Team | 2人で支払い分担を事前に決める。1人`$500`想定 |
| 6/26候補 | Dailyなら`$250`、フリーロールなら無料 |
| キャッシュ | `$1/$2`想定。座る額や損失上限は各自で決める |

WSOPのオンライン登録ではカード手数料がかかる場合がある。公式登録ページでは、Visa/Mastercard/Discover/Amex利用時に追加3%の記載があるため、最終的な支払い方法は現地/WSOP+で確認する。

## 申告ライン

| 場面 | 申告ライン | 今回の方針 |
| --- | ---: | --- |
| 日本出国/入国 | 100万円相当超 | USD現金は`$6,000`未満に抑える |
| 米国入国/出国 | `$10,000`超 | 持ち込まない |

米国の`$10,000`は、現金だけでなくトラベラーズチェック、マネーオーダー、一定の小切手等も含む。家族などグループで旅行している場合、合計額で判定されることがある。日本側も現金・小切手・有価証券等の合計で100万円相当超なら申告対象。

## 両替・外貨宅配

| 方法 | 向いている場合 | 注意 |
| --- | --- | --- |
| 外貨宅配 | 出発前に余裕を持って準備したい、レート比較したい | 配送日数、送料、最低金額、本人確認、受取方法を確認 |
| 空港両替 | 当日受け取りで安心したい | レートは事前比較。混雑時は時間に余裕を見る |
| 銀行/街中両替 | 近くに店舗がある | 営業時間、米ドル在庫、手数料を確認 |
| 現地ATM | 不足時の予備 | カジノATMは手数料が高い可能性。常用しない |

おすすめは、出発1-2週間前に外貨宅配または国内店舗で`$1,800-$2,200`程度を用意し、不足時だけカード/ATMで補う形。外貨宅配はレートと手数料を比較しやすいが、郵送の受取日数と本人確認があるため直前には向かない。

## 現地でのルール

- ポーカー資金と生活費を混ぜない。
- キャッシュの損失上限は各自で決める。
- 酒を飲んだらキャッシュの追加Buy-inをしない。
- ATMで下ろす前に、まず残りの現金と翌日の予定を確認する。
- 盗難/紛失時に備えて、カード会社の停止方法をメモしておく。
