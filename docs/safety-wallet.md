# 保険・カード・緊急控え

保険、カード停止、緊急連絡を出発前に整えるための公開用メモ。実際の証券番号、カード番号、個別電話番号、本人情報はGitLabに書かない。

## 目的

- 米国で医療、盗難、カード紛失が起きても、最初の連絡先を迷わない。
- 片方のスマホが使えなくても、もう片方と紙控えで動ける。
- 公開版には「保存済みか」「どこを見るか」だけを残す。

## 作る非公開控え

| 控え | 入れる内容 | 公開版に書くこと |
| --- | --- | --- |
| 保険メモ | 保険会社、証券番号、緊急窓口、キャッシュレス診療の使い方 | 保険控えは非公開で保存済み |
| カード停止メモ | カード会社名、停止窓口、アプリ導線、予備カード | 停止窓口は非公開控えと紙へ保存 |
| ツアー緊急メモ | 最終案内の緊急連絡先、空港集合、ホテル情報 | ツアー緊急連絡先は最終案内の非公開控えで確認 |
| 紙控え | 最小限の連絡導線、保険、カード停止、空港集合 | 紙控えを2人で分散して持つ |

## 出発前チェック

| チェック | 完了条件 |
| --- | --- |
| 保険の補償 | 医療、救援、携行品、航空機遅延の範囲を確認 |
| キャッシュレス診療 | 使い方、対象地域、最初に連絡する窓口を確認 |
| 証券番号 | GitLab外の非公開控えと紙に保存 |
| カード停止 | 各カードの停止導線をアプリ/電話/紙で確認 |
| 予備カード | メイン財布と別の場所に分ける |
| Emergency現金 | 日常財布と分け、使わない予備として持つ |
| ツアー緊急 | 最終案内が来たら非公開控えへ保存 |
| 2人確認 | 2人とも非公開控えと紙の場所を知っている |

## 非公開控えの作り方

1. `private/`や`local-private/`ではなく、各自のスマホ/クラウド/紙で持つ場所を決める。
2. 保険、カード、ツアー緊急、空港集合を1つの非公開メモにまとめる。
3. 紙控えは2セット作り、1人だけに集約しない。
4. スマホのオフライン保存を確認する。圏外、電池切れ、アプリ未ログインでも開けるかを見る。
5. 公開版には「保存済み」「非公開控えを見る」だけ反映する。

## 2人で確認する5分ミーティング

| 確認 | 合格ライン |
| --- | --- |
| 片方のスマホだけで全予約を開けるか | Grand Canyon、Sphere、ツアー最終案内、保険、カード停止導線を開ける |
| 紙控えの場所 | 2人がそれぞれ持ち、片方の荷物紛失でも残る |
| カード停止 | どのカードを止め、どの予備カードで動くか説明できる |
| 医療判断 | 911、保険相談、ホテル/施設スタッフ相談の順番を説明できる |
| 公開版 | GitLabに実番号、氏名、QR、カード情報がない |

## 現地で困った時

| 状況 | 最初にやること |
| --- | --- |
| 命、事故、犯罪 | 米国の緊急通報は911 |
| 病院へ行くか迷う | 保険の非公開控えから相談/キャッシュレス診療窓口を見る |
| 財布をなくした | 相方へ共有し、カード停止導線を開く |
| スマホをなくした | 相方の端末、紙控え、ホテル/施設Lost and Foundで動く |
| ツアー集合に遅れそう | 予約確認メールまたは最終案内の非公開控えから連絡 |

### 財布・カード紛失の順番

1. その場から離れず、最後に使った場所を相方とメモする。
2. パスポート、スマホ、残った現金、残ったカードを確認する。
3. 非公開控えから該当カードの停止導線を開く。
4. 不正利用通知を確認し、必要ならカード会社に連絡する。
5. その日のポーカー、買い物、追加観光を止める。
6. 予備カード、Emergency現金、ホテル、空港移動だけを優先する。

### 医療・体調不良の順番

1. 命、事故、犯罪なら911。
2. 迷う場合は保険の非公開控えから相談/キャッシュレス診療窓口を見る。
3. ホテル、カジノ、Sphere、空港なら近くのスタッフに助けを求める。
4. 予定は削る。Grand Canyon、Sphere、帰国移動に影響する場合は早めに縮小する。

## 紙に書く最小項目

紙にはカード番号全面やメール本文を印刷しない。

公開してよい行動カードは [paper-backup-card.md](paper-backup-card.md) を使う。実番号は印刷後に手書きするか、別の非公開紙で持つ。

| 紙に書く | 書かない |
| --- | --- |
| 保険会社名、証券番号、緊急窓口 | カード番号全面 |
| カード会社名、停止導線 | 暗証番号 |
| ツアー最終案内の緊急導線 | 予約確認メール全文 |
| ホテル名、空港集合情報 | チケットQR/バーコード |

## 公開版への反映

公開版に反映してよい書き方:

- `保険控えは非公開で保存済み`
- `カード停止窓口は非公開控えと紙で確認`
- `Emergency現金は日常財布と分ける`
- `ツアー緊急連絡先は最終案内の非公開控えで確認`

公開版に書かない:

- 保険証券番号。
- カード番号、暗証番号。
- 個別契約に紐づく緊急電話番号。
- 本人名、住所、生年月日。
