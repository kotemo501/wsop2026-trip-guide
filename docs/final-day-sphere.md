# 6/27 Sphere + 最終日導線

6/27(土)は14:00のThe Wizard of Oz at Sphereを軸にする。夜に空港へ移動する日なので、ポーカー、長時間の買い物、終了時刻が読めない予定は入れない。

## 結論

| 項目 | 内容 |
| --- | --- |
| 公演 | The Wizard of Oz at Sphere |
| 日時 | 2026-06-27(土) 14:00 |
| 開場 | 13:15目安。Sphere公式FAQでは開演45分前にdoors open |
| 遅刻 | Late seating不可。13:15-13:30には入場列に入る |
| チケット | Mobile tickets only。事前にスマホのWalletへ追加 |
| 支払い | Sphere内はcashless |
| バッグ | no bag寄り。大きいバッグ、バックパック、スーツケースは不可前提 |
| 荷物預かり | Sphereはbags/coats/personal belongingsのcheck-inを提供しない |
| 最終判断 | 荷物はHorseshoe Bell Deskまたはツアー指定場所に預け、Sphereへは身軽に行く |

購入前に見るカード: [sphere-ticket-purchase-card.md](sphere-ticket-purchase-card.md)

荷物と空港集合の専用カード: [final-day-luggage-airport-card.md](final-day-luggage-airport-card.md)

## 持って行くもの

| 持つ | 理由 |
| --- | --- |
| スマホ | モバイルチケット、地図、連絡 |
| モバイルバッテリー | チケット表示と帰国導線の生命線 |
| クレカ/デビット | Sphereはcashless |
| パスポート | 帰国日なので手元管理 |
| 薄い上着 | 館内冷房対策 |

持たないもの:

- スーツケース。
- 大きいバックパック。
- 買い物袋を増やしすぎる予定。
- 預けられない貴重品をホテル荷物に入れること。
