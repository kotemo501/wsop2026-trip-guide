## ポーカー予算感

Buy-in、キャッシュゲーム用資金、食事・観光費は分けて考える。ここではポーカーだけの目安を置く。

### 1人あたりのトーナメント予算

| 予算感 | 内容例 | 合計目安 | 向いている人 |
| --- | --- | ---: | --- |
| 軽め | `$250` Daily Deepstack x 1、フリーロール | `$250` | 記念に1本だけ打ちたい |
| 標準 | `$250` Daily Deepstack x 1、`$400` Daily Deepstack x 1、フリーロール | `$650` | 低額中心にWSOP会場で複数回打ちたい |
| Tag Team込み | Tag Teamを2人で割る想定、`$250` or `$400` Daily x 1、フリーロール | `$750-$900` | 2人旅らしい本命体験を入れたい |
| 多め | Tag Team割り勘、`$250` Daily x 1、`$400` Daily x 1、サテライト少額 | `$1,000-$1,200` | ポーカー比重を高めたい |
| 本戦追加 | `$1,500` Freezeout、Tag Team割り勘、Daily x 1 | `$2,250-$2,400` | ブレスレットイベントを複数入れたい |

Tag Teamの`$1,000/team`は、2人で割るなら1人`$500`想定。

### キャッシュゲーム用の別枠

| 予算感 | 内容例 | 1日上限 | 総額目安 |
| --- | --- | ---: | ---: |
| 軽め | 低ステークスを短時間、負けたら終了 | `$200-$300` | `$300-$600` |
| 標準 | 2-3回座る、時間固定 | `$300-$500` | `$600-$1,000` |
| 多め | 複数日しっかり打つ | `$500-$1,000` | `$1,500+` |

キャッシュゲームはBuy-inと損失上限を別に決める。例: `$1/$3`に`$300`で着席しても、追加Buy-inしないならその日の上限は`$300`。

### 今回のおすすめ予算案

| 案 | 内容 | 1人あたり目安 |
| --- | --- | ---: |
| 低予算で楽しむ | `$250` Daily x 1、フリーロール、キャッシュ少し | `$550-$850` |
| バランス | Tag Team割り勘、`$250` or `$400` Daily x 1、フリーロール、キャッシュ少し | `$1,000-$1,500` |
| ポーカー多め | Tag Team割り勘、`$250` Daily、`$400` Daily、キャッシュ2-3回 | `$1,500-$2,200` |
| 本戦も追加 | Freezeout、Tag Team割り勘、Daily、キャッシュ少し | `$2,500-$3,000` |

まずは1人あたり「ポーカー総額上限」を`$1,000`、`$1,500`、`$2,000`のどれにするか決めると、候補を絞りやすい。
