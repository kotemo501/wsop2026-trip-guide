## 結論

公開用しおりは、友人へ共有してスマホで使える状態。Grand Canyonは予約済みで、当日集合・支払い・持ち物の導線も公開版に反映済み。残りは「友人担当待ち」「現地確認」「付帯保険確認」「可変運用」に分けてHOLD管理する。

出発前の完全GOとは分ける。公開しおりとしては使えるが、出発直前は [go-no-go-check.md](go-no-go-check.md) と [departure-readiness.md](departure-readiness.md) で、非公開控え、保険/カード、荷物、チケット表示を確認する。
