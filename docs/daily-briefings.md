# 毎朝ブリーフィング

現地で朝に読む添乗メモ。今日の目的、固定時刻、持つもの、避けることだけを見る。予約番号、参加者名、チケットQR、保険証券番号、カード番号は公開版に書かない。

## 共通ルール

- 朝に時間がない時は、先に [today-command.md](today-command.md) で今日の3アクションだけ見る。
- 表が読みづらい時は、スマホ縦読み用の [daily-concierge.md](daily-concierge.md) を先に見る。
- 今日の固定予定を先に見る。
- パスポート、スマホ、クレカ、現金、モバイルバッテリー、水を確認する。
- 翌日に重い予定がある日は、夜のポーカーや遠出を控える。
- 別行動する前に、再集合場所と時刻を決める。
- 迷ったら [pocket-itinerary.md](pocket-itinerary.md) と [travel-support-playbook.md](travel-support-playbook.md) を開く。
