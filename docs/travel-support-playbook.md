# 現地サポート・プレイブック

公開できる範囲で、旅行代理店の現地サポートメモのように使うための行動手順。予約番号、個人名、連絡先、保険証券番号、カード番号は書かない。必要な番号は各自のメール、Wallet、紙の控え、非公開メモで持つ。

## 使い方

焦っている時は、先に [sos-quick-card.md](sos-quick-card.md) で最初の60秒だけ確認する。

| タイミング | 見る場所 | やること |
| --- | --- | --- |
| 毎朝 | 今日の行動チェック | 集合時刻、財布、カード、ID、水、充電を確認 |
| 予定変更時 | 判断ルール | 帰国便、体力、予約済み予定を優先して崩さない |
| 困った時 | トラブル別手順 | 先に安全確保、次に相方へ共有、最後に予約先/保険/カード会社 |
| 公開前 | 公開版の扱い | 個人情報と予約票をGitLabへ置かない |

## 毎朝3分チェック

| 確認 | 内容 |
| --- | --- |
| 今日の固定予定 | 時刻、場所、移動時間、キャンセル不可の予定を確認 |
| 相方との合意 | 別行動するなら再集合場所と時刻を決める |
| 交通 | 徒歩/Uber/Lyft/ツアーバスのどれで動くか決める |
| 財布 | 日常財布は必要分だけ。残り現金は分散 |
| カード | クレカ2枚の場所、暗証番号、利用通知を確認 |
| ID | パスポート原本が必要な日か、コピー/写真で足りる日か確認 |
| 電池 | スマホ、モバイルバッテリー、ケーブル |
| 水分 | 水、カフェイン、軽食。特に6/25は多め |

## 判断ルール

| 状況 | 優先する判断 |
| --- | --- |
| 眠い、体調が微妙 | トーナメント追加より休息。翌日の固定予定を守る |
| 予定が押している | 予約済み予定、集合時刻、帰国便に関係する予定を優先 |
| ポーカーで勝ち進んだ | 6/25 Grand Canyonと6/27帰国導線に影響する場合は、事前合意なしに無理をしない |
| 予算が不安 | キャッシュゲームと買い物を止める。食事は軽食寄りに戻す |
| 別行動したい | 再集合時刻、場所、連絡不能時の待機期限を決めてから分かれる |

## トラブル別手順

### 集合に遅れそう

1. 相方へすぐ共有する。
2. 予約済みツアーなら、非公開の予約確認メールまたは公式サイトから連絡導線を開く。
3. Grand Canyonは05:05 pickupなので、04:55時点で集合場所にいない場合は緊急扱いにする。
4. 連絡がつかない場合、相方は集合場所で待ち続けすぎず、予約先の指示を優先する。

### スマホをなくした

1. 相方のスマホで現在地と最後に使った場所をメモする。
2. Find My / Googleの端末検索を試す。
3. ホテル、カジノ、Sphere、ツアー会社など、その日に通った施設のLost and Found導線を確認する。
4. チケットや予約票は相方の端末、紙控え、メールで代替できるようにしておく。

### 財布・カードをなくした

1. まず相方へ共有し、最後に使った場所を確定する。
2. カード会社アプリまたは停止窓口で該当カードを止める。
3. 残りカードと現金を確認し、その日の予定を縮小する。
4. ポーカー、買い物、追加観光は止め、帰国導線と食事を優先する。

### 体調不良

1. 水分、冷房、日差し、睡眠不足を疑い、まず涼しい場所で休む。
2. Grand Canyon、Sphere、空港移動に影響する場合は早めに予定を縮小する。
3. 医療が必要なら保険の非公開控えからキャッシュレス診療/相談窓口を確認する。
4. 片方が動けない場合、もう片方は無理に単独観光を増やさず、ホテル拠点に戻す。

### Grand Canyon当日

| 時刻 | 行動 |
| --- | --- |
| 前夜 | Horseshoe Rideshare Lot、Flamingo Road側North Entrance、Real Bodies Exhibit近くを歩いて確認 |
| 04:15 | 起床。水、軽食、上着、カード、ID、予約確認を準備 |
| 04:55 | 集合場所に到着して待機 |
| 05:05 | pickup |
| 入場時 | 非米国居住者fee `$100/人` をnon-cashで支払えるようにする |
| 帰着時 | drop-offがHorseshoeでない場合、ホテルまでの徒歩/配車を判断 |

### Sphere + 最終日

1. チェックアウト後、大きい荷物はホテルBell Deskまたはツアー指定場所へ預ける。
2. Sphereには大きいバッグを持ち込まない。
3. 14:00公演なら13:15開場目安。モバイルチケットとカードを確認する。
4. 終演後は写真や買い物を引っ張りすぎず、荷物回収と空港集合を優先する。
5. 空港集合時刻が確定したら、この資料と [final-day-sphere.md](final-day-sphere.md) に反映する。

## 公開版と非公開控えの分担

| 公開版に書いてよい | 非公開で持つ |
| --- | --- |
| 集合時刻、集合場所、持ち物、追加支払いの種類 | 予約番号、参加者名、確認メール本文 |
| 日別スケジュール、予算目安、候補リスト | パスポート番号、ESTA番号、保険証券番号 |
| 公式サイトURL、問い合わせ前の確認項目 | カード番号、カード停止窓口、個人の電話番号 |
| トラブル時の一般手順 | 予約票PDF、チケットQR、本人確認書類画像 |
