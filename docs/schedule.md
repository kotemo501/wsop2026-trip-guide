# 日別スケジュール案

時刻は特記なき限りラスベガス現地時間です。フライト時刻はチャーター便のため、最終日程表で要確認です。

## 使い方

- `固定`: ツアーやフライトで動かしにくい予定。
- `本命`: 今回の旅の目的に合いやすい候補。
- `差し替え`: 体力、勝ち進み、予算で入れ替える候補。
- `重い`: 予約・移動・時間拘束が大きい候補。

候補の詳細は [options.md](options.md) を参照。
