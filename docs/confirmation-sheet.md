| 預け荷物重量 | 1個30kg以下、3辺203cm以下が一般ルール |
| チャーター便で条件変更ありそうか | あり得る。ツアー最終案内を優先 |
| モバイルバッテリー | 機内持込。予備リチウム電池/パワーバンクは預け不可 |
| 液体類 | 機内持込は100ml以下容器、透明袋。大きい液体は預け荷物 |
| 帰りのお土産重量余裕 | 行きの預け荷物は20-23kg程度に抑えると余裕がある |
| メモ | 水は重いので預け荷物に大量投入しない。旅行用ケトルを持つなら預け荷物へ |

確認先:

- ZIPAIR手荷物PDF: https://www.zipair.net/service/baggage/departures_after_October_29_2023_en.pdf
- ZIPAIR危険物リスト: https://www.zipair.net/service/baggage/rule/Dangerous%20Goods%20List%20and%20Label%20%28English%29.pdf

荷物メモ:

- パスポート、現金、カード、薬、スマホ、充電器、モバイルバッテリーは機内持込。
- カップ麺、衣類、液体類の予備、刃物/爪切り系は預け荷物。
- モバイルバッテリーは容量表示が読めるものを持つ。100Wh以下なら通常扱いやすい。

水のコスパ:

| 方法 | 価格感 | 判断 |
| --- | ---: | --- |
| 日本から水を持参 | 500ml x 10本で約5kg | 重量を食うので非推奨 |
| 成田の保安検査後に購入 | 空港価格 | 機内用に1-2本ならあり |
| 空ボトル持参 + 給水 | ほぼ無料 | 一番軽い。給水場所があれば有力 |
| ZIPAIR機内購入 | 有料。無料水は期待しない | 買えるが、品切れや注文タイミングの不便あり |
| ラスベガスCVS/Walgreens | 24本で`$3-$8`台目安 | 滞在用は現地まとめ買いが最有力 |

結論:

- 行きの機内用: 空ボトル + 保安検査後に1-2本購入。
- 現地用: 初日にCVS/Walgreensで24本パックまたは大きいボトルを買う。
- ホテル/カジノ内の単品水は`$6-$10+`になり得るので避ける。

## 9. 節約食の現実性

通常食は節約し、体験にお金を回す方針。
カップ麺と旅行用ケトルとかを持っていく方針かな。水の入手も考えないと。ホテルでもらえるのかとか調べられる？？

調査結果: CVS/Walgreensの24本パックはセール込みで`$3-$8`台の例がある。ホテル/カジノ内の単品水はかなり高いので、初日まとめ買いが良い。

| 項目 | 回答 |
| --- | --- |
| Horseshoe客室に冷蔵庫があるか | 口コミ/FAQでは部屋タイプにより小型冷蔵庫ありの例。全室確約ではないため要確認 |
| 電子レンジが使える場所 | 客室内は基本なし想定。ホテル共用/フードコートで使えるかは未確認 |
| お湯を入手できる場所 | 客室ケトルは期待しない。旅行用ケトル持参が現実的 |
| 近くのCVS/Walgreens等 | Horseshoe内/近くにCVS Pharmacyあり。3645 Las Vegas Blvd S、24時間営業情報あり |
| 初日買い出し場所 | CVS at Horseshoe / 近隣CVS・Walgreens |
| カップ麺用の箸/フォーク | 日本から割り箸/使い捨てフォーク持参推奨 |
| 水の購入方法 | ホテルでもらえる前提にしない。CVS/Walgreensで24本パックか大きめボトルを買う |
| メモ | 旅行用ケトルは便利だが機内持込重量を圧迫。預け荷物へ。電圧対応を確認 |

買う候補:

- 水
- カップ麺
- プロテインバー/ナッツ
- バナナなど軽食
- コーヒー/カフェイン
- 日焼け止め

節約食メモ:

- ラスベガスのホテル客室はケトルなしのことが多いので、カップ麺方針なら折りたたみ/小型の海外電圧対応ケトルを持つ。
- 水はホテル売店だと高くなりやすい。初日にCVSでまとめ買いする。
- 冷蔵庫がなければ、常温保存できる水、カップ麺、バー、ナッツ中心にする。

水の価格メモ:

| 場所 | 価格目安 | メモ |

## 10. 聖地巡礼ご飯

中華とラーメンを候補にする。通常食は節約。
中華・ラーメンの値段帯が欲しい。ヨコサワの動画でよく出てるやつ

方針: 聖地巡礼ご飯は現地予算に含める。通常食は節約し、外食はラーメン1回 + 中華1回までを基本にする。
