# 60秒SOSカード

現地で焦った時に最初の60秒だけ見るカード。予約番号、参加者名、確認メール本文、保険証券番号、カード番号、個人の電話番号は書かない。必要な番号は非公開控え、メール、Wallet、紙のミニカードを見る。

## まず60秒

1. 安全な場所へ移動する。
2. 相方に状況、場所、次にすることを送る。
3. 予約、保険、カードの実番号が必要なら非公開控えを開く。
4. 命、事故、犯罪なら`911`。

## 分岐

### 集合に遅れそう

- 相方へ現在地と到着見込みを送る。
- Grand Canyonやツアー系は非公開の予約確認メールから会社へ連絡する。
- Grand Canyonは04:55時点で集合場所にいなければ緊急扱い。
- 相方は集合場所で待ち続けすぎず、予約先やツアー側の指示を優先する。

### スマホ紛失/電池切れ

- 相方の端末で現在地と最後に使った場所をメモする。
- Find My / Googleの端末検索を試す。
- ホテル、カジノ、Sphere、ツアー会社のLost and Foundを確認する。
- 紙のミニカードと相方の端末で予約・集合・空港導線を代替する。

### 財布/カード紛失

- 最後に使った場所を1つに絞る。
- 非公開控えからカード停止導線を開く。
- 残りカード、Emergency現金、パスポートの有無を確認する。
- ポーカー、買い物、追加観光は止め、食事と帰国導線を優先する。

### 体調不良

- 水、冷房、日陰、座れる場所を確保する。
- 予定を1つ削る。Grand Canyon、Sphere、空港移動に影響するなら早めに縮小する。
- 医療が必要なら保険の非公開控えから相談/キャッシュレス診療窓口を見る。
- 片方が動けない時は、ホテル拠点に戻す判断を優先する。

### 最終日が押した

- Sphere後は写真、買い物、寄り道より荷物回収を優先する。
- 空港集合時刻が近いなら、観光を切って移動する。
- 荷物を持ったままSphereへ行かない。
- ツアー最終案内が公開版と違う場合、ツアー最終案内を優先する。
