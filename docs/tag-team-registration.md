## 登録導線

1. Horseshoe Las Vegas / Paris Las VegasのWSOP会場へ行く。
2. `WSOP Registration`と`Verification`の案内板を探す。分からなければスタッフに聞く。
3. Caesars Rewards Cardがない/見つからない場合は、先にfront deskまたはreprint kioskで対応する。
4. WSOP LIVEでアカウントが開けるか確認する。
5. 現地登録なら、パスポート、Caesars Rewards、支払いを出してEvent #66 Tag Teamを指定する。
6. オンライン登録済みなら、初回はWSOP Verificationで本人確認を完了し、アプリのticketまたは印字ticketを確認する。
7. 2人とも同じチームで登録されているか、テーブル/席、開始時刻、Late Registration、交代ルールを確認する。

## 交代ルールの要点

- 1つのチームで1つのスタックを使う。
- チームメンバーは登録終了までにチェックインし、少なくとも1周のブラインドをプレイする必要がある前提で動く。
- アクティブなハンド中は交代しない。
- チーム内で交代タイミングを事前に決めておく。
- 賞金は公式ルール上、チーム内で均等配分される扱いを前提にする。
