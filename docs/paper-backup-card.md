# 紙のミニカード

スマホ紛失、電池切れ、通信不良、アプリ障害に備えて印刷する最小カード。公開版なので、予約番号、参加者名、確認メール本文、チケットQR、パスポート/ESTA番号、保険証券番号、カード番号は書かない。実番号は印刷後に手書きするか、別の非公開紙で持つ。

## 使い方

- A4で1枚だけ印刷する。
- 2人で同じ紙を1枚ずつ持つ。
- 実番号を書く場合は、GitLab上ではなく印刷後に手書きする。
- 紛失リスクがあるので、カード番号全面やメール本文は書かない。
