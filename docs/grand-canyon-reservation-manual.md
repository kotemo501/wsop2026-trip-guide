## 2026年非米国居住者feeの結論

2026年1月1日以降、Grand Canyon National Parkでは、16歳以上の非米国居住者に追加fee`$100/人`がかかる。NPSのFAQでは、commercial tour groupの参加者にもこのfeeが必要とされている。

今回の2人は日本在住者なので、原則として2人で`$200`を別予算に入れる。

| 論点 | 結論 |
| --- | --- |
| feeは払う必要があるか | 必要。commercial tourでも対象 |
| GC Tours `$70`に含まれるか | 含まれない前提で見る。`$70/人`はツアー本体として安すぎ、NPSの`$100/人`を内包しているとは考えにくい |
| 確認すべきこと | 予約時にGC Toursが`$100/人`を追加徴収するか、当日公園側/ツアー会社に払うか |
| 予算への入れ方 | GC Toursなら`$70 x 2 + $100 x 2 + tip/軽食`で最低`$340+`を見る |
| もしツアー画面で`nonresident fee included`と出た場合 | その表示をスクショ保存し、総額に含めて扱う |

予約画面や確認メールに`NPS nonresident fee`、`additional entrance fee`、`not included`などの記載がない場合でも、`$100/人`は別途かかる前提で進める。

## 予約手順

### 1. 候補ページを開く

候補ページを2-3個開き、同じ条件で比較する。

比較する項目:

| 項目 | 書き込む内容 |
| --- | --- |
| サイト名 | 例: GrandCanyon.com / GC Tours / Scenic / VegasTours |
| ツアー名 | 正式名称 |
| 日付 | 2026-06-25(木) |
| 人数 | 2 |
| 1人料金 | 表示価格 |
| 総額 | 税・手数料込みの最終額 |
| pickup | Horseshoeか、近隣ホテルか |
| 集合時刻 | 例: 6:15am / 6:40am |
| 集合場所 | 例: North Entrance Tour Lobby |
| 帰着/drop-off | National Park ExpressはPark MGM等に限定される可能性あり |
| 昼食 | 込み / voucher / 別 |
| 入園料 | 込み / 別 |
| 非米国居住者 fee | 込み / 別 / 不明 |
| キャンセル | 期限と返金条件 |

### 2. 日付・人数を入れる

予約カレンダーで`June 25, 2026`を選ぶ。日付が現地時間基準であることを確認する。

人数は`2 Adults`。参加者情報を後で求められる場合は、2人分のローマ字名を入れる。

### 3. Pickupを選ぶ

Pickup欄で、次の優先順に選ぶ。

| 優先 | 選択 |
| --- | --- |
| 1 | Horseshoe Las Vegas |
| 2 | Bally's / Horseshoe |
| 3 | Paris Las Vegas pickup at Horseshoe |
| 4 | Caesars / Flamingo / Planet Hollywoodなど徒歩圏 |
| 5 | 指定ターミナル集合。早朝Uberが必要なら最後の候補 |

Horseshoeの場合、集合場所は`North Door`、`North Entrance`、`Tour Lobby`、`facing Flamingo Road`などの表記が出ることがある。予約後に地図と写真で場所を確認する。

### 4. 追加オプションを外す

基本はバスツアー本体だけでよい。次の追加は原則つけない。

| オプション | 方針 |
| --- | --- |
| IMAX | 時間と体力を使うので不要寄り |
| Helicopter | 予算が大きく上がるため今回対象外 |
| Hummer / Jeep | South Rimバス目的から外れるなら不要 |
| Meal upgrade | 標準昼食があるなら不要 |
| Photo package | 現地判断。予約時は不要 |

### 5. 総額を確認する

支払い直前の画面で、以下を確認する。

- 2人分の合計金額。
- 税・手数料・燃油サーチャージ。
- Grand Canyon National Park入園料。
- 2026年の非米国居住者 fee `$100/人`の徴収方法。
- 昼食、飲料水、軽食の有無。
- チップが含まれるか。含まれないなら当日現金を用意する。

非米国居住者 feeの徴収方法が画面上で不明なら、予約前に問い合わせる。問い合わせる場合は次の文でよい。

```text
We are non-U.S. residents from Japan and plan to join the Grand Canyon South Rim tour on June 25, 2026. How will the 2026 NPS nonresident fee of $100 per person be collected: during booking, before the tour, or on the day of the tour?
```

### 6. キャンセル条件を確認する

最低限、次をメモする。

| 項目 | メモ |
| --- | --- |
| 無料キャンセル期限 | 例: 24/48/72時間前 |
| 変更期限 | 日付・人数・pickup変更がいつまで可能か |
| no-show扱い | 集合遅れ時に返金されるか |
| 悪天候/道路事情 | 代替、返金、日程変更の扱い |

### 7. 支払いを完了する

支払い前の最終確認:

- 日付が`Thu, June 25, 2026`になっている。
- 人数が2人。
- South Rim / Grand Canyon National Park行き。
- PickupがHorseshoeまたは徒歩圏。
- 合計金額が予算内。
- キャンセル条件を読んだ。

支払い後、確認メールを保存する。
