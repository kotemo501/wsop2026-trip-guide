## 食事候補の考え方

| シーン | 候補方針 |
| --- | --- |
| 到着日 | 予約なし、ホテル近く、軽め |
| トーナメント前 | 重すぎない、会場近く、移動少なめ |
| 勝った日/記念日 | 1回だけ少し良い店を予約 |
| 最終日 | 空港移動前に時間が読みやすい店 |

## 友人と決める食事テーマ

- ステーキに行くか
- ビュッフェに行くか
- 日本食/アジア系を1回入れるか
- 予約する高めの店を1つだけ決めるか
- ポーカー中の軽食・水・カフェインをどうするか
