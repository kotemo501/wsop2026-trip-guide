# 現地予算

1人あたりの現地予算メモです。ツアー代は支払い済みなので、この資料では現地で追加で使うドル建て費用だけを管理します。

換算は計算しやすいように`$1 = 160円`で置く。2026-05中旬のUSD/JPYはおおむね157円台なので、手数料・両替・カードレート込みの見積もりとして160円で見る。

まずは現地予算`$2,700`を基本線にする。キャッシュゲームやお土産を厚めにする場合は各自で上振れを見る。
