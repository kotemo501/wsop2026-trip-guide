# 毎朝マネーカード

現地で朝に見るお金の運用カード。ポーカー用、生活用、Emergencyを混ぜないためのルールだけを書く。実際の持参額、財布の場所、カード番号、暗証番号、停止窓口はGitLabに書かない。

## 毎朝30秒

1. 今日の固定支払いを確認する。
2. Daily walletに入れる分だけ持つ。
3. ポーカー資金と生活費を別にする。
4. Emergency現金は使わない前提で分散保管する。

## 財布の分け方

| 財布/封筒 | 目安 | 使う場面 | ルール |
| --- | ---: | --- | --- |
| Daily wallet | `$100-$200` | 水、軽食、チップ、近場移動 | その日使う分だけ |
| Poker固定 | `$750` | 6/23 Daily + 6/24 Tag Team割り勘 | 生活費と混ぜない |
| Cash Game | `$900-$1,200` | `$1/$3`を300-400BB見る | 損失上限を決めてから座る |
| Tips/交通 | `$200-$300` | ツアーチップ、Uber/Lyft、軽食 | 小額紙幣を優先 |
| Emergency | `$300-$500` | 紛失、カード停止、移動トラブル | 普段は使わない |

## 日別ルール

| 日 | 財布に入れるもの | 止めるライン |
| --- | --- | --- |
| 6/22 | Daily wallet、クレカ、少額チップ | 到着日からキャッシュで粘らない |
| 6/23 | Daily `$250`、生活用少額、上着 | 眠い/疲れたら粘らない |
| 6/24 | Tag Team割り勘分、生活用少額 | 翌朝04:15起床に響く追加予定なし |
| 6/25 | カード、チップ少額、生活用少額 | 非米国居住者fee `$100/人`はnon-cash |
| 6/26 | Daily wallet、クレカ、生活用少額 | 疲労が残るなら予定を増やさない |
| 6/27 | クレカ、生活用少額、空港移動用 | 買い物より荷物回収と空港集合 |

## キャッシュゲーム停止ルール

- 酒を飲んだら追加Buy-inしない。
- 眠い時は打たない。
- 翌日が早い日は打たない。
- 決めた損失上限に達したら終了。
- 生活費やEmergencyから追加しない。

## カードを使う場面

| 場面 | 方針 |
| --- | --- |
| Grand Canyon非米国居住者fee | non-cash支払いに対応できる状態にする |
| WSOPトーナメント | cashでも払える状態にする。カード/WSOP LIVEは初回verificationと手数料を確認 |
| Sphere | チケット、入場、現地支払いはカード中心 |
| ホテル周辺/買い物 | クレカ中心。利用通知を見られる状態にする |
| ATM | 不足時の予備。カジノATMは手数料が高い可能性 |

## カードの持ち分け

| 種類 | 朝の確認 |
| --- | --- |
| メインカード | 今日使う財布に入っている。利用通知と暗証番号を確認済み |
| 予備カード | 日常財布とは別の場所。紛失時に残る |
| ATM/デビット | 使う前に手数料、限度額、残高を確認 |

カード番号、停止窓口、予備カードの保管場所は公開版に書かない。必要なら [safety-wallet.md](safety-wallet.md) の非公開控えを見る。

## 紛失時

1. 相方へ共有する。
2. 残った現金、カード、パスポートを確認する。
3. 非公開控えからカード停止導線を開く。
4. ポーカー、買い物、追加観光を止める。
5. 食事、ホテル、空港移動を優先する。
