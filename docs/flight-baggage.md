## MPC / ESTA / チャーター便

### やること

1. ESTAの有効期限とパスポート番号を確認する。
2. MPCアプリを入れて、対応する範囲でパスポート情報を準備する。
3. 運航会社名をツアー最終案内で確認する。
4. その運航会社がCBPのVWP signatory carrierか、旅行会社または航空会社に確認する。
5. MPCレーンが使えない場合は通常入国に並ぶ前提で動く。

### 判断

| 質問 | 現時点の扱い |
| --- | --- |
| MPCは使う？ | 入れておく。LASは対応空港。ただし当日レーン運用次第 |
| MPCはESTAの代わり？ | 代わりではない。ESTAやパスポートは別に必要 |
| 日本国籍ESTAで使える？ | CBPはapproved ESTAを持つreturning Visa Waiver Program travelerを対象に含めている |
| 初回米国入国でも使える？ | 「returning」とあるため、初回なら通常入国前提にする |
| チャーター便で注意することは？ | VWP/ESTAで搭乗するには、運航会社がVWP signatory carrierであることを確認する |

## 機内持ち込み

ZIPAIRの約款では、無料で持ち込める機内持込手荷物は合計2個まで、2個合計7kg以下とされています。サイズ目安は以下。

| 個数 | サイズ目安 |
| --- | --- |
| 1つ目 | 40 x 25 x 55cm |
| 2つ目 | 35 x 25 x 45cm |

7kgを超えそうな場合は、合計15kgまでの重量追加枠を事前に確認します。当日空港で焦らないよう、出発前に自宅で重さを測る。

## 預け荷物

受託手荷物は、1個あたり30kg超、または3辺合計203cm超は受け付け不可の前提で見る。

注意:

- パスポート、現金、カード、薬、電子機器は預けない。
- 帰りのお土産分を考え、行きの時点で重さに余裕を残す。
- チップ、ポーカー用現金、緊急用現金は必ず手元管理。

## 機内に入れるもの

- パスポート。
- 現金、クレカ、海外ATM用カード。
- 薬、コンタクト、メガネ。
- スマホ、充電器、モバイルバッテリー。
- 1日分の着替え。
- 羽織りもの。
- イヤホン。
- 重要な予約確認メールを見られる端末。

## 預けるもの

- 衣類。
- 洗面具。
- 液体類の予備。
- 刃物類。
- 爪切りなど、機内持込で止められそうなもの。

## 持ち込み禁止・注意物

| 種類 | 方針 |
| --- | --- |
| 刃物、切るための道具 | 機内持込不可。預け荷物へ |
| スポーツ用具 | 機内持込にしない |
| 花火、ガス缶、武器類 | 基本NG |
| 予備リチウム電池、モバイルバッテリー | 預けず機内へ |
| 注射器・注射針 | 必要書類と申告を確認 |
| 液体類 | 機内持込は100ml以下容器。大きい液体は預け荷物へ |

モバイルバッテリーは預け荷物に入れない。容量表示が読める状態にして、端子がショートしないようケースや袋に分ける。

## ラスベガス旅の荷造り方針

行き:

- 機内持込は7kgに収める。
- ポーカー用現金やカードは分散。
- 現地で買える水や軽食は持ち込みすぎない。

帰り:

- お土産分の重量を残す。
- 液体土産は預け荷物へ。
- 現金・カード・電子機器は最後まで手元。

## 出発前チェック

- [ ] 機内持込2個のサイズを確認する。
- [ ] 機内持込の合計重量を測る。
- [ ] 預け荷物が30kg以下か確認する。
- [ ] モバイルバッテリーを預け荷物に入れていないか確認する。
- [ ] パスポート、現金、カード、薬を機内持込に入れる。
- [ ] ESTAの有効期限、パスポート番号、滞在先を確認する。
- [ ] MPCアプリを入れる。
- [ ] チャーター便の実運航会社がVWP signatory carrierか確認する。
- [ ] ツアー最終案内でチャーター便の手荷物条件を確認する。
