- 予約番号、予約票、参加者名は公開版に載せない。各自のメール/Wallet/ローカル保存で確認。
- 6/28 02:35 LAS発なので、6/27夜の空港移動に影響する予定は入れない。
- 疲れている日はポーカーより睡眠。6/25 Grand Canyon前夜と最終日は特に無理しない。

## 困った時

| 状況 | 最初にやること |
| --- | --- |
| 集合に遅れそう | まず相方に連絡。予約系は非公開の予約確認メールから会社へ連絡 |
| 財布/カード紛失 | カード停止、予備カード、Emergency現金。現金は同じ財布に集約しない |
| 暑さ/疲労 | 水、日陰、冷房、休息を優先。予定を削る |
| 最終日が押した | 寄り道より荷物回収と空港集合を優先 |
| 緊急 | 米国の緊急通報は911 |
