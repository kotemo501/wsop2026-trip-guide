# 体力・休憩カード

現地で「まだ遊ぶか、休むか」を決めるカード。旅行を楽しくするために、睡眠、水、食事、暑さ、翌日の固定予定を先に見る。医療が必要な状態なら、このカードではなく保険の非公開控えや`911`を優先する。

## 朝の判定

| 状態 | 判定 | 今日の方針 |
| --- | --- | --- |
| よく眠れた、足が軽い | Green | 予定どおり。ただし夜の上限は決める |
| 眠い、足が重い、食欲が弱い | Yellow | 予定を1つ削る。近場と軽食中心 |
| 頭痛、強い疲労、胃腸不調、暑さでだるい | Red | ポーカー/観光追加なし。ホテル拠点で回復 |

迷ったらYellow。Grand Canyon前夜、Grand Canyon翌日、最終日は無理しない。

## 毎日やること

- 水を持つ。
- 冷房のある場所で休む時間を作る。
- 食事を抜きすぎない。
- カフェインで眠気をごまかして長時間打たない。
- 翌日の固定予定を確認してから夜の予定を決める。

## 食事の最低ライン

| 場面 | 最低ライン |
| --- | --- |
| 朝 | 水、軽食、カフェイン少量 |
| トーナメント前 | 重すぎない食事、上着、水 |
| Grand Canyon | 水、軽食、上着、酔い止め候補 |

## 予定を削る順番

1. 追加キャッシュゲーム。
2. 予約していない観光。
3. 遠い買い物。
4. 長い外食。
5. 固定予定、帰国導線、予約済み予定は最後まで守る。
